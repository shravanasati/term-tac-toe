# Term Tac Toe

A terminal-based multiplayer Tic-Tac-Toe game with real-time WebSocket gameplay and minimal, photonic design.

## Quick Start

### Installation

```bash
pip install term-tac-toe
```

### Run

```bash
term-tac-toe
```

To connect to a self-hosted server:

```bash
term-tac-toe --server <server-ip>
```
