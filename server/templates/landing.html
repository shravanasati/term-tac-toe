Term Tac Toe

A terminal-based multiplayer Tic-Tac-Toe game that keeps the vibe minimal, photonic, and impossible to ignore.

Installation

1.

Install the client from PyPI or using pip inside this repository.

pip install term-tac-toe
2.

Run the client and point it at a live server IP.

uv run term-tac-toe --server <server-ip>

Term Tac Toe stays true to the terminal experience while allowing friends to pair up for real-time play. The server endpoint you are viewing now hosts the React-free, websocket-powered game loop. Start it, share the room code, and let the board glow.
